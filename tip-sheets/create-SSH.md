    "# A Beginner’s Guide to Setting Up SSH Keys for GitHub\n",
    "\n",
    "GitHub authentication has evolved over time, and passwords are no longer\n",
    "considered secure enough for daily Git operations.\n",
    "\n",
    "SSH (Secure Shell) allows you to securely connect and authenticate to GitHub\n",
    "without repeatedly entering your credentials.\n",
    "\n",
    "An SSH key acts like a digital identity for your machine.\n",
    "\n",
    "Your machine keeps a **private key** (secret), and GitHub stores a **public key**\n",
    "(safe to share).\n",
    "\n",
    "\n",
    "\n",
    "---\n",
    "\n",
    "## Step 1: Open Git Bash\n",
    "\n",
    "Open **Git Bash** on your computer.\n",
    "\n",
    "---\n",
    "\n",
    "## Step 2: Check if an SSH Key Already Exists\n",
    "\n",
    "Before creating a new key, check if your system already has one.\n",
    "\n",
    "```bash\n",
    "ls ~/.ssh\n",
    "```\n",
    "\n",
    "If the folder doesn’t exist, it simply means no SSH key has been created yet.\n",
    "This is normal and expected behaviour. Proceed to Step 3.\n",
    "\n",
    "If the `.ssh` folder already exists, you may see:\n",
    "\n",
    "- `id_ed25519` (Private Key)\n",
    "- `id_ed25519.pub` (Public Key)\n",
    "\n",
    "You may reuse the existing key or create a new one.\n",
    "\n",
    "---\n",
    "\n",
    "## Step 3: Generate a New SSH Key\n",
    "\n",
    "Create a new SSH key using the following command:\n",
    "\n",
    "```bash\n",
    "ssh-keygen -t ed25519 -C \"your_email@example.com\"\n",
    "```\n",
    "\n",
    "When prompted for the file location, press **Enter** to use the default.\n",
    "\n",
    "You may also set a passphrase for extra security, but this step is optional.\n",
    "\n",
    "Once the key is generated, the `.ssh` folder will be created automatically.\n",
    "\n",
    "Check again by running:\n",
    "\n",
    "```bash\n",
    "ls ~/.ssh\n",
    "```\n",
    "\n",
    "You should see:\n",
    "\n",
    "- `id_ed25519`\n",
    "- `id_ed25519.pub`\n",
    "\n",
    "Never share your private key (`id_ed25519`).\n",
    "\n",
    "---\n",
    "\n",
    "## Step 4: Start the SSH Agent\n",
    "\n",
    "The SSH agent allows Git to use your key.\n",
    "\n",
    "```bash\n",
    "eval \"$(ssh-agent -s)\"\n",
    "```\n",
    "\n",
    "You should see output similar to:\n",
    "\n",
    "```\n",
    "Agent pid 1234\n",
    "```\n",
    "\n",
    "---\n",
    "\n",
    "## Step 5: Add the SSH Key to the Agent\n",
    "\n",
    "Run the following command:\n",
    "\n",
    "```bash\n",
    "ssh-add ~/.ssh/id_ed25519\n",
    "```\n",
    "\n",
    "If you set a passphrase, you will be prompted to enter it now.\n",
    "\n",
    "---\n",
    "\n",
    "## Step 6: Copy the Public SSH Key\n",
    "\n",
    "Run:\n",
    "\n",
    "```bash\n",
    "cat ~/.ssh/id_ed25519.pub\n",
    "```\n",
    "\n",
    "Copy the entire line that starts with `ssh-ed25519`.\n",
    "\n",
    "---\n",
    "\n",
    "## Step 7: Add the SSH Key to GitHub\n",
    "\n",
    "1. Go to github.com\n",
    "2. Click your profile photo → **Settings**\n",
    "3. Click **SSH and GPG keys**\n",
    "4. Click **New SSH key**\n",
    "5. Fill in:\n",
    "   - **Title:** My Windows Laptop\n",
    "   - **Key type:** Authentication Key\n",
    "   - **Key:** Paste the copied key\n",
    "6. Click **Add SSH key**\n",
    "7. Enter your GitHub password if prompted\n",
    "\n",
    "Your SSH key will now appear in your GitHub account.\n",
    "\n",
    "---\n",
    "\n",
    "## Step 8: Test the SSH Connection\n",
    "\n",
    "Run:\n",
    "\n",
    "```bash\n",
    "ssh -T git@github.com\n",
    "```\n",
    "\n",
    "If asked to continue connecting, type `yes`.\n",
    "\n",
    "A success message will appear:\n",
    "\n",
    "```\n",
    "Hi <username>! You've successfully authenticated...\n",
    "```\n",
    "\n",
    "Congratulations! SSH is working.\n"
